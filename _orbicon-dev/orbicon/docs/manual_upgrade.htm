Beginning the manual upgrade

You should now be at the stage where you can upload the Orbicon framework files to your server. Initiate an FTP connection to the server and navigate to the directory of the Orbicon framework you wish to upgrade.

Step-by-step upgrade

  1. Create backup directory, for example 20070606
  2. Create temporary directory, for example new
  3. Unpack files from compressed archive into temporary directory new
  4. Move orbicon directory and index.php from root to backup directory 20070606
  5. Move orbicon directory and index.php from temporary directory new to root
  6. Validate upgrade
  7. If the update succeded, archive backup directory to 20070606.tar.gz and remove temporary directory new
  8. If the update failed, perform rollback from backup folder 20070606

Once all the files are uploaded, you should verify that everything went fine by logging in into the system.
