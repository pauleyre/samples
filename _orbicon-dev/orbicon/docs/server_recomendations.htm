Server Recommendations

  • Windows operating system, Linux or another Unix based operating system

  • Apache 1.3.x or 2.0.x

  • PHP 5.2.0 or higher with FTP extension and with the following set in php.ini:
    • the max_input_time directive set to a value of at least 30.
    • the post_max_size and upload_max_filesize directives set to the size of the largest documents you wish to be able to upload.
    • the memory_limit directive set to at least 8M (16M or 32M recommended for large image processing).
    • the max_execution_time directive set to at least 15.
    • the register_globals directive set to Off.
    • safe mode disabled, or PHP suExec enabled.

  • MySQL 4.0.27 or higher

  • PHP extension GD 2.0 or higher

  • XPDF for Linux
