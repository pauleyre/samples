Template marks

Template marks are special text tags that serve as identifiers for various system rendering tasks. They are exclusively used in HTML templates and outside of them have no use or power.

As you can see template marks <!>DOMAIN_NAME and <!>ORBX_SITE_URL have special meaning to the system and will be replaced by appropriate data during layout rendering. Some template marks are always available to end-users, some are part of their modules and some are user-generated (such as boxes). Below is a comprehensive list of available template marks.

User-generated boxes
Depends on user input (for example: <!>MY_RED_BOX) Displays a user generated-content All Example Not Available
Custom RSS boxes
Depends on user input (for example: <!>MYRSS_HTTP-RSS-CNN-COM-RSS-CNN_TECH-RSS) Displays a box with simply formatted RSS content All See RSS channels

Template marks are case-sensitive. Typing <!>poll will not work.
