Installing

Assuming the previous step went well, you should now have the Orbicon framework files on your server with their appropriate permissions set. All that is needed now is to configure your MySQL database and admin account.

  • MySQL server name: This is where we specify the location of the MySQL database.. 99% of the time this will be localhost.
  • MySQL username: This is the Username for your MySQL database. Often you will have to set this up in your web host's control panel. If in doubt, consult your host.
  • MySQL password: This is the password for your MySQL database. Often you will have to set this up in your webhost's control panel. If in doubt, consult your host.
  • MySQL Database name: This is the name of your MySQL database. Often you will have to set this up in your webhost's control panel. If in doubt, consult your host.

Once you have filled in all of the settings, you can proceed and fill in the details of your admin account; this will create your username on the site. Once you have filled in all the data, click on proceed.

Your Orbicon framework is now installed and you can browse to it.
