Beginning the automatic upgrade

You should now be at the stage where you can upgrade Orbicon framework using the Update Manager. To access Update Manager (located under System section) you must login into Orbicon with appropriate privileges. Once you are ready to proceed, click the Update button and wait for the update process to end. The update may take a while so you shouldn't interrupt it by closing the page.

Once the update has finished, you should read the update's message log and verify that everything went well.
