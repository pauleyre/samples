Desktop

Desktop is system's front page. The main menu is positioned on the top of the screen. Most menu buttons have their own submenus which are displayed when mouse cursor is hovered over them. Submenus can be closed by clicking anywhere.

Buttons "Desktop", "Languages", "Help" and "Exit" are always available while access to other sections depends on your privileges. Accessing any option from the submenu will display option's window from where we can control its functionalities. (Each window has two panels, workspace on the left, settings and tools sidebar on the right)

System's front page is also a desktop.

This desktop is unique to each user and can be modified by its owner. Every system window has its own icon on the left which can be added or removed from desktop by clicking the icon


The icon is on the desktop and can be removed by clicking green checkmark.


The icon is not on the desktop and can be added by clicking the the red X.

Besides icons, your desktop can read and display RSS feeds and show current time.
