Predefined constants

A number of predefined constants is available for use within system's scope. Developers should use these instead of failing back to general PHP constants were possible

Constant Description
DOC_ROOT Path to Orbicon framework's root directory. Use this instead of $_SERVER['DOCUMENT_ROOT']
ORBX_CLIENT_IP Client's IP address. Use this instead of $_SERVER['REMOTE_ADDR']
ORBX_USER_AGENT Client's User Agent. Use this insted of $_SERVER['HTTP_USER_AGENT']
DOC_3RD_PARTY Path to Orbicon framework's 3rd party root directory
SCHEME Scheme for current site (http or https)
DOMAIN Domain name for current site
DOMAIN_NO_WWW Same as DOMAIN but stripped of any www prefix
ORBX_SITE_URL Full URL to Orbicon framework's base address
