Columns

"Columns" (which are available under menu "Content") enables us to create and organize horizontal and vertical menus, boxes and internal pages. If we open "Columns" we can see that the system allows us to select which menu we wish to edit. We could try to add 5 pagest to horizontal menu. We select the horizontal menu graphic, type in the page's title on the right sidebar and submit it. Once we are finished with all 5 pages, we can rearrange their order with " drag and drop".
