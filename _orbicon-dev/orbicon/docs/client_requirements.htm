Client Requirements

Minimum Browser Requirements:

  • Microsoft Internet Explorer 6.0 for Windows
  • Opera 9
  • Mozilla 1.3
  • Firefox 1.0
  • Netscape 7.0
  • SeaMonkey 1.1

These versions of browsers support both AJAX and Rich-Text editing. While the system may load with other or older browsers, functionality cannot be guaranteed

Browser Settings Requirements:

  • Javascript enabled for many functions
  • Java Software (Java Runtime Environment) enabled for Drag and Drop Upload functionality
