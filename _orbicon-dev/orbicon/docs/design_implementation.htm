Issue: Where should I upload my website's favicon?
Solution: Favicon should be located in site/gfx directory.

Issue: How can I switch between different languages?
Solution: Links such as www.mywebsite.com/?ln=en or www.mywebsite.com/?ln=de will switch between English and German language. For a full list of ISO codes please refer to ISO 639 code list. Remember that the language must first be installed in the system. Below is an example.
