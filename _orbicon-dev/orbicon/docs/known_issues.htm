Known issues

This is a compilation of known issues and their workarounds.

Issue: Rich-Text Editor does not respond to keyboard input in Firefox.
Solution: Press Spacebar on your keyboard. This will enable keyboard input for Firefox.

Issue: Interaction with active content (Flash, etc.) is disabled in Internet Explorer.
Solution: This is a known issue for users of latest Internet Explorer 6 or better. Orbicon comes packaged with Adobe's Flash Player Detection Kit which offers solution for this issue. Refer to their documentation for help on implementing. The Detection Kit is located in orbicon/3rdParty/ac directory.

Issue: Executing SQL query with foreign characters through mysql shell, corrupts those same characters.
Solution: Execute the following SQL query SET NAMES utf8;. For more information, visit http://dev.mysql.com/doc/refman/4.1/en/charset-connection.html

Issue: Double and single quotes inside my data becomes escaped. For example " is saved as \"
Solution: If your web server is Apache, you can create .htaccess file with the following content: php_flag magic_quotes_gpc off
