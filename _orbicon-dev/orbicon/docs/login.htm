Login

Orbicon framework does not utilize a visible link to its authorization screen. To bring up this screen, access key X must be pressed. This depends on your browser.

  • Internet Explorer 5+ (PC): Hold down the ALT key, press the X key, release both keys then press ENTER

  • Internet Explorer 4 (PC): Hold down the ALT key and press the X key

  • Internet Explorer 5+ (Mac): Hold down the CTRL key and press the X key

  • Internet Explorer 4.5 (Mac): Access keys are not supported

  • Netscape 6 and earlier (PC and Mac): Access keys are not supported

  • Netscape 7, Firefox, Mozilla (PC): Hold down the ALT key and press the X key

  • Firefox, Mozilla (Mac): Hold down the CTRL key and press the X key

  • SeaMonkey, Firefox 2+ (PC): Hold down the Shift key and press ALT key, then press the X key

  • Firefox 2+ (Mac): Hold down the Shift key and press CTRL key, then press the X key

  • Safari and Omniweb (Mac): Hold down the CTRL key and press the X key

  • Opera (PC and Mac): Hold down the Shift key and press Escape, release both keys, then press the X key
