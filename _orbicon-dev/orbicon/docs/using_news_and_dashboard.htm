Using news dashboard

After adding documents to the system, we can use them in our news entries. Under menu "Content" we select submenu "News dashboard". On the left panel, we can see the list of news from all categories. To add a new entry we must click "Add new" (located on the bottom).

The newly opened window, entitled "News", is ready to create new news entry. A news entry is composed of various properties but the most important ones are title, lead text, full text entry, publish date, and published status. To use an article previously created in Texts database, we must navigate to that text using "Database browser" on the right sidebar.

We can also select our lead text image this way.

After selecting texts and optionally, an image for news, we must select publish date and set published status to "Yes"

Note: News can be published with dates set in future.
