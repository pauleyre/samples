Module building example

In this example, we'll create a very simple module that displays "Hello world!" in a stylized box. We'll also allow the user to use Javascript to change that text into something more interesting.

First we create a directory for our module - hello-world in the modules directory. Once we have done that, we should create the neccessary mod.ini file and directory languages.

We'll add English language translation immediately as seen in the example below.

en.php

Once we are done, the file should be saved under module's languages directory with name en.php.

Next, we'll create our mod.ini file as seen below.

mod.ini

Next we'll write our module's content into hello.world.php (This is the file we configured in our mod.ini)


hello.world.php's content will be displayed as a box in place of <!>HELLO_WORLD template mark as we configured it in mod.ini
