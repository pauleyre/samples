Managing text content

Open menu "Databases", and select "Texts" from submenu. The window "Texts" opens. On the right sidebar we can see that the card entitled "Database browser" is opened. This is the database for all text documents. Since this is our first usage of the system we will be prompted to create our first text category. We create our first category by typing the name and submitting it. (We can see our newly created category on the right.) Now we can create a text article. We enter text's title, for example "my article" and submit it. The system saves text and brings up the advanced text editor. Inside text editor we can write our article's content, save it or modify it.

What have we accomplished so far? We created text database category and inside of it we have created text "my article". If we wish to add additional text to our category, we simply type in another article title and submit it.
