Obtaining package for manual upgrade

While there are more than a few different packages available in your Pickup Depot, the one you want is the upgrade package.

Unzip the package, ensuring you keep the directory structure. If you are using Windows XP, the easiest way is to right click on the file you downloaded and select the "extract all" option and follow the wizard. If you are using another operating system, then use WinZip or a similar program.
