3rd party resources

Orbicon framework comes packaged with 3rd party resources which are easily available for use and implementation. These resources are stored in folder orbicon/3rdParty.

Name Description Directory name (in orbicon/3rdParty)
Flash Player Detection Kit The Flash Player Detection Kit helps developers implement robust player detection in JavaScript environment. For more information please visit http://www.adobe.com/products/flashplayer/download/detection_kit/ ac
CSS parser This PHP class is able to parse CSS files making the properties available for examination and further processing. For more information please visit http://www.phpclasses.org/browse/package/1289.html css
EditArea A free JavaScript editor for source code. For more information please visit http://www.cdolivet.net/index.php?page=editArea edit_area
IP to Country IP2Country provides functions to retrive real user's IP address, and to know the country of any IP. For more information please visit http://www.codewalkers.com/c/a/Miscellaneous-Code/IP2Country/ iplist
MagpieRSS MagpieRSS provides an XML-based (expat) RSS parser in PHP. For more information please visit http://magpierss.sourceforge.net/ magpierss
Rad Upload Lite With Rad Upload's drag and drop functionality, users can transfer files to server just as easily as copying from one folder to another. For more information please visit http://www.radinks.com/upload/ rad
SafeHTML - Anti-Cross Site Scripting (XSS) HTML parser SafeHTML' is an anti-XSS HTML parser. It strips all potentially dangerous content within HTML: opening tags without closing tags, closing tags without opening tags, certain attributes (on*, data*, dynsrc, javascript:/vbscript:/about:, etc.), protocols, expressions or behavior in styles, and any other active content. For more information please visit http://pixel-apes.com/safehtml/ safehtml
script.aculo.us - web 2.0 javascript A collection of Web 2.0 style JavaScript libraries that help web developers add AJAX functionally to projects. For more information please visit http://script.aculo.us/ scriptaculous
TAR File Manager A PHP Implementation of the TAR (Tape-Archive) Format most commonly used on UNIX Operating systems. This class works on both Windows and UNIX for reading and writing TAR files. It even supports Gzipped TAR files if you have the ZLIB PHP module installed on your server. For more information please visit http://www.phpclasses.org/browse/package/529.html tarmanager
TCPDF TCPDF is an Open Source PHP class for generating PDF files on-the-fly without requiring external extensions. For more information please visit http://www.tecnick.com/public/code/cp_dpage.php?aiocp_dp=tcpdf tcpdf
Yahoo! UI Library (YUI) The Yahoo! User Interface (YUI) Library is a set of utilities and controls, written in JavaScript, for building richly interactive web applications using techniques such as DOM scripting, DHTML and AJAX. For more information please visit http://developer.yahoo.com/yui/ yui
